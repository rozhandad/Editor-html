پاراگراف ها یا Paragraphs در HTML

حرف P از کلمه پاراگراف ( Paragraph ) گرفته شده و کاملا از اسمش مشخصه که چیه! یعنی یک متن / پاراگراف به همین سادگی.

ما اگه بخوایم یه متن رو بزاریم داخل صفحه وبمون خب خیلی راحت حتی بدون استفاده از تگ خاصی میتونیم اینکارو انجام بدیم.

برای مثال یه متن گذاشتیم داخل تگ Body :

1
2
3
4
5
<body>
 
این یک متن هست بدون هیچ تگ خاصی و بدون مشکل در خروجی نشونش میده
 
</body>

امتحان کنید

ولی این اصلا بهینه و منطقی نیست، و از دید مرورگرها و موتورهای جستجوگر اینکار کار کثیفی هست، چرا که در زبان HTML برای هرکاری و هرچیزی تگ مخصوص وجود داره، پس برای بحث بهینه بودن ما باید همیشه سعی کنیم از تگ های مناسب در مکان مناسب استفاده کنیم.

در زبان HTML میگه آقا اگه خواستی عکس بزاری تو صفحه، خب از تگ img استفاده کن، خواستی جدول بزاری از تگ table استفاده کن و همینطور تا بره، پس داره میگه که هرتگی برای کاری درست شده.

برای متن گذاشتن هم همینطور ، داره میگه از تگ P استفاده کنید، تگ P یعنی پاراگراف ینی پاراگرافی از یک متن به همین سادگی، پس ما میتونیم برای قرار دادن متن ها از تگ <p> استفاده نماییم.

شکل کلی برای استفاده از تگ پاراگراف در HTML بصورت زیر می باشد :

1
<p>In Yek Paragraph Ast</p>

به راحتی در ابتدا تگ شروع را با استفاده از <p> ایجاد مینماییم، سپس محتوای مورد نظر را وارد مینماییم و در نهایت با استفاده از <p/> تگ پایانی را بسته و پاراگراف ایجاد خواهد شد.

امتحان کنید

بصورت پیش فرض تگ P دارای یکسری دستورات CSS می باشد، ( البته تقریبا میشه گفت همه ی تگ های HTML دارای CSS پیش فرض هستن ) یعنی مثلا اگه به مثال بالا توجه کنید یه فاصله مشخصی بین پاراگراف ها وجود داره ، این میشه همون دستورات CSS که میگم بصورت پیش فرض وجود داره و مرورگرها بصورت پیش فرض از این دستورات استفاده میکنند.

اصلا نگران نباشید، ان شاالله کمی جلوتر که با CSS آشنا شدید یاد میگیرید که چجوری میشه همین دستورات پیش فرض رو تغییر و یا حذف کرد. ( مثلا الان یه فاصله مشخصی بین تگ های P وجود داره ولی خب شاید من بخوام این فاصله رو زیاد و یا کمترش کنم و.. )
